# omniforge

Omniforge is a multi-genre creative forge designed to ignite inspiration and empower producers to craft their next record from a powerful foundation. Built to evolve across genres, workflows, and creative ecosystems.

## Dark Mythic Theme

This repository now includes a **dark mythic** UI theme concept with:

- Deep charcoal foundation layers
- Electric blue accents and restrained glow lighting
- Minimal, modern component styling
- Soft neon hover states for interactive buttons

### Files

- `theme.css` — reusable theme tokens and component styles
- `preview.html` — standalone preview of the theme

To view the preview locally, open `preview.html` in a browser.
